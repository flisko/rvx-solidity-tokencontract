
# installing
```
cd rvx-solidity-tokencontract
npm install

cd static
npm install



```

# usage
```
cd rvx-solidity-tokencontract
nodemon npm start


cd static
npm start

```

# deployment
before deployment:
```
cd rvx-solidity-tokencontract
nodemon npm start


cd rvx-solidity-tokencontract/static
yarn build
npm install -g serve
serve -s build


```
